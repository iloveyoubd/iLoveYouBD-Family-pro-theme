# 📱 ILoveYouBD Neon Pro - অ্যান্ড্রয়েড অ্যাপ বিল্ড গাইড (Android App Build Guide)

এই প্রোজেক্টে **Capacitor (by Ionic)** ইন্টিগ্রেট করা হয়েছে। এর মাধ্যমে আপনার তৈরি করা এই নিয়ন থিমের দুর্দান্ত সব ফিচার (যেমন- `মায়া চ্যাটবট`, `এনআইডি মেকার`, `লাইভ টিভি`, `এআই ক্রু` ইত্যাদি) কোনো পরিবর্তন ছাড়াই সরাসরি একটি প্রফেশনাল অ্যান্ড্রয়েড অ্যাপে রূপান্তর করা যাবে।

নিচে ধাপে ধাপে সহজ গাইড দেওয়া হলো, যার মাধ্যমে আপনি আপনার নিজের কম্পিউটার ব্যবহার করে অ্যাপটি জেনারেট (.apk) করতে পারবেন।

## 🛠️ প্রয়োজনীয় সফটওয়্যার (Prerequisites)
শুরু করার আগে আপনার কম্পিউটারে নিচের সফটওয়্যারগুলো ইনস্টল থাকতে হবে:
1. **Node.js** (LTS ভার্সন)
2. **Android Studio** (লেটেস্ট ভার্সন) এবং Android SDK

## 🚀 অ্যান্ড্রয়েড প্রোজেক্ট রান ও বিল্ড করার ধাপসমূহ

### ধাপ ১: আপনার লোকাল পিসিতে প্রোজেক্ট ডাউনলোড ও সেটআপ করা
প্রথমে এই প্রোজেক্টটি জিপ হিসেবে এক্সপোর্ট বা ডাউনলোড করে আপনার কম্পিউটারে আনজিপ করুন। এরপর কমান্ড প্রম্পট (Terminal) ওপেন করে প্রজেক্ট ফোল্ডারে প্রবেশ করুন এবং রান করুন:
```bash
npm install
```

### ধাপ ২: ওয়েব ফাইল বিল্ড করা
অ্যাপের রিঅ্যাক্ট (React) ওয়েব ভার্সনটিকে বিল্ড করার জন্য নিচের কমান্ডটি চালান:
```bash
npm run build
```
এটি প্রোজেক্টের ভিতরে `dist` ফোল্ডার তৈরি করবে, যেখানে মূল অ্যাপ ফাইলগুলো থাকবে।

### ধাপ ৩: অ্যান্ড্রয়েড প্ল্যাটফর্ম যুক্ত করা
প্রথমবারের মতো আপনার প্রোজেক্টে অ্যান্ড্রয়েড বিল্ডার যোগ করতে নিচের কমান্ডটি রান করুন:
```bash
npm run cap:add
```
*এই কমান্ডটি রান করার সাথে সাথেই প্রোজেক্টের ভেতর ফাইলগুলো প্রফেশনাল অ্যান্ড্রয়েড অ্যাপ স্ট্রাকচারে রূপান্তর হয়ে যাবে এবং একটি `android` ফোল্ডার তৈরি হয়ে যাবে।*

### ধাপ ৪: অ্যাপ সিঙ্ক (Sync) করা
ভবিষ্যতে আপনি যখনই কোড বা ডিজাইনে কোনো পরিবর্তন করবেন, তখন অ্যান্ড্রয়েড অ্যাপের সাথে সিঙ্ক করতে এই হ্যাসেল-ফ্রি স্ক্রিপ্টটি রান করবেন:
```bash
npm run cap:sync
```

### ধাপ ৫: অ্যান্ড্রয়েড স্টুডিওতে অ্যাপটি ওপেন করা
নিচের কমান্ডটি দিলে আপনার তৈরি সম্পূর্ণ অ্যান্ড্রয়েড প্রোজেক্টটি সরাসরি **Android Studio**-তে ওপেন হয়ে যাবে:
```bash
npm run cap:open
```

## 📲 অ্যান্ড্রয়েড স্টুডিও (Android Studio) থেকে APK জেনারেট করা

অ্যান্ড্রয়েড স্টুডিও ওপেন হওয়ার পর গ্র্যাডল সিঙ্ক (Gradle Sync) হতে ১-২ মিনিট সময় লাগতে পারে।

### ১. নিজের ফোনে সরাসরি টেস্ট করার জন্য:
- আপনার ফোনের **Settings > Developer Options** থেকে **USB Debugging** চালু করুন।
- ইউএসবি ক্যাবল দিয়ে ফোনটি কম্পিউটারের সাথে কানেক্ট করুন।
- Android Studio-এর ওপরের মেনু থেকে আপনার ফোনটি সিলেক্ট করুন এবং সবুজ রঙের **Run (Play)** বাটনে ক্লিক করুন। সরাসরি আপনার ফোনে অ্যাপটি ইনস্টল হয়ে চালু হবে!

### ২. শেয়ার করার জন্য APK ফাইল ডিরেক্ট জেনারেট করতে:
- Android Studio-এর ওপরের মেনুবার থেকে ক্লিক করুন: **Build > Build Bundle(s) / APK(s) > Build APK(s)**
- বিল্ড প্রসেস শেষ হলে নিচে একটি পপআপ আসবে যেখানে **"locate"** লিখা থাকবে। সেখানে ক্লিক করলেই আপনি ইনস্টল করার উপযোগী `.apk` ফাইলটি পেয়ে যাবেন!

### ৩. গুগল প্লে স্টোরে রিলিজ করার জন্য (.aab ফাইল):
- ওপরের মেনুবার থেকে ক্লিক করুন: **Build > Generate Signed Bundle / APK...**
- **Android App Bundle** সিলেক্ট করুন এবং Next চাপুন।
- নতুন একটি Keystore (সার্টিফিকেট কী) তৈরি করুন, পাসওয়ার্ড দিন এবং **Release** অপশন সিলেক্ট করে ফিনিশ করুন। এটি আপনার জন্য গুগল প্লে স্টোরে পাবলিশ করার মতো ১০০% প্রফেশনাল `.aab` ফাইল জেনারেট করে দেবে!

## 🎨 অ্যাপ আইকন (App Icon) এবং স্প্ল্যাশ স্ক্রিন (Splash Screen) পরিবর্তন
মাস্টার ক্লাসের মতো অ্যাপ আইকন ও বুটিং স্ক্রিন প্রফেশনাল করতে:
- আপনার অ্যাপের লোগো (`icon.png`) এবং স্প্ল্যাশ স্ক্রিনের ব্যাকগ্রাউন্ড ইমেজের জন্য একটি ইমেজ ফোল্ডারে রাখুন।
- খুব সহজে কমান্ড লাইনে আইকন রেন্ডার করতে নিচের পপুলার টুলটি ব্যবহার করতে পারেন:
  ```bash
  npx cordova-res android --skip-config --copy
  ```
  এটি আপনার ফোল্ডার থেকে স্বয়ংক্রিয়ভাবে সব অ্যান্ড্রয়েড স্ক্রিন সাইজের উপযোগী আইকন ও স্প্ল্যাশ স্ক্রিন রিসাইজ করে `android/app/src/main/res/` ফোল্ডারে ইনজেক্ট করে দিবে!

## 🔐 অ্যান্ড্রয়েড পারমিশন (Permissions)
আপনার অ্যাপের কিছু ফিচার (যেমনঃ এনআইডি মেকারে কাস্টম ফটো আপলোড করা) ব্যবহারের জন্য ক্যামেরা ও মেমোরি অ্যাক্সেস প্রয়োজন হতে পারে। এর জন্য `android/app/src/main/AndroidManifest.xml` ফাইলে এই পারমিশনগুলো অটোমেটিক্যালি হ্যান্ডেলড থাকে। কোনো নির্দিষ্ট ফিচারের জন্য এক্সট্রা পারমিশনের প্রয়োজন হলে জাস্ট নিচের লাইনগুলো মেনশন করতে পারেন:
```xml
<uses-permission android:name="android:permission.CAMERA" />
<uses-permission android:name="android:permission.READ_EXTERNAL_STORAGE" />
