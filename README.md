## 🇧🇩 পরিচিতি (Introduction)
**ILoveYouBD Neon Pro** একটি অত্যন্ত চমৎকার ও প্রফেশনাল মাল্টি-ফাংশনাল হাইব্রিড প্ল্যাটফর্ম। সম্পূর্ণ ডার্ক এবং নিয়ন আরজিবি (RGB Neon Layout) থিমে তৈরি এই অ্যাপ্লিকেশনটিতে ব্যবহারকারীদের জন্য আকর্ষণীয় সব টুলস এবং চ্যাটবট ইন্টিগ্রেশন রয়েছে। এটি একই সাথে ব্রাউজার এবং অ্যান্ড্রয়েড ডিভাইসে স্মুথলি রান করার জন্য অপ্টিমাইজড। 

এই রিপোজিটরিতে অ্যাপের সম্পূর্ণ সোর্স কোড এবং ল্যাটেস্ট মোবাইল ফ্রেমওয়ার্ক **Capacitor** যুক্ত করা আছে, যা দিয়ে আপনি আপনার নিজের ব্রান্ডিং এ সরাসরি প্লে-স্টোর রেডি APK তৈরি করতে পারবেন।

## 🎨 মূল ফিচারসমূহ (Premium Core Features)

আমাদের অ্যাপ্লিকেশনে ব্যবহারকারীদের জন্য যুক্ত করা প্রতিটা মডিউল অত্যন্ত যত্নসহকারে ডিজাইন করা হয়েছে:

### 1. 🤖 মায়া চ্যাটবট (Maya Chatbot AI)
- **স্মার্ট অ্যাসিস্ট্যান্ট**: রিয়েল-টাইম প্রশ্ন-উত্তরের জন্য একটি চাবক ও বুদ্ধিমান এআই চ্যাটবট।
- **নাইট মুড ও নিয়ন বাবল**: সুন্দর মেসেজিং বাবল এবং আকর্ষণীয় নিয়ন ইফেক্ট সমৃদ্ধ ইন্টারফেস।

### 2. 🪪 এনআইডি কার্ড মেকার (NID Maker Utility)
- **ইনস্ট্যান্ট আইডি রিসোর্স**: কাস্টম নাম, ফটো, স্বাক্ষর এবং অন্যান্য তথ্য দিয়ে ফ্রি এবং প্রফেশনাল আইডি জেনারেট করার টুল।
- **ডাউনলোড সাপোর্ট**: লাইভ প্রিভিউ শেষে এক ক্লিকে হাই-কোয়ালিটি কার্ড ফাইল মেমোরিতে সেভ করার সুবিধা।

### 3. 📺 লাইভ টিভি (Live TV Streamer)
- **সবচেয়ে জনপ্রিয় চ্যানেল**: রিয়েল-টাইম লাইভ ক্যাটাগরি অনুসারে জনপ্রিয় বাংলাদেশি ও আন্তর্জাতিক স্যাটেলাইট টিভি চ্যানেল সরাসরি স্ট্রিম করার মিডিয়া প্লেয়ার।

### 4. 👥 এআই ক্রু (AI Crew / AI Workspace)
- **উন্নত গ্রুপ ওয়ার্কস্পেস**: মাল্টিপল ক্যাটাগরির এআই রোবট একসাথে অ্যাক্টিভ করে কন্টেন্ট জেনারেশন ও রিসার্চ ডেভেলপমেন্ট করার মডিউল।

### 5. 💬 কমিউনিটি প্রশ্নোত্তর (Community Q&A Forum)
- **সোশ্যাল ফিড**: সাধারণ প্রশ্ন ও উত্তর এবং আইডিয়া শেয়ারিং করার জন্য একটি রিয়েল-টাইম কমিউনিটি প্ল্যাটফর্ম।

### 6. ⚙️ অ্যাডমিন প্যানেল (Powerful Admin Dashboard)
- **কন্ট্রোল সেন্টার**: সম্পূর্ণ ওয়েবসাইট ও ওয়ান-ক্লিক সেটিংস রিফ্যাক্টরিং করার প্রফেশনাল ড্যাশবোর্ড।

### 7. 🌈 আরজিবি নিয়ন বর্ডার (RGB Glow Theme)
- **ফিজিক্যাল গেমিং ভাইব**: সম্পূর্ণ স্ক্রিন জুড়ে একটি গতিশীল ও গর্জিয়াস আরজিবি নিয়ন বর্ডার অ্যানিমেশন, যা মোবাইল স্ক্রিনে দুর্দান্ত দেখায়।

## 📂 প্রোজেক্টের প্রধান ডিরেক্টরি পরিচিতি (Project File Structure)

আপনার গিটহাবের এই প্রোজেক্ট ফোল্ডারে থাকা গুরুত্বপূর্ণ ফাইল এবং ফোল্ডারগুলোর উদ্দেশ্য নিচে দেওয়া হলো:

```bash
├── 📁 src/                     # মূল সোর্স কোড (রিঅ্যাক্ট ফ্রন্টঅ্যান্ড)
│   ├── 📁 components/          # সবগুলো প্রফেশনাল উইজেট ও মডিউল
│   │   ├── 📄 MayaChatbot.tsx   # মায়া চ্যাটবট ইন্টারফেস
│   │   ├── 📄 NIDMaker.tsx      # এনআইডি মেকার টুল
│   │   ├── 📄 LiveTV.tsx        # লাইভ টিভি প্লেয়ার
│   │   ├── 📄 AICrew.tsx        # এআই ক্রু মডিউল
│   │   ├── 📄 RGBBorder.tsx     # আরজিবি এনিমেটেড বর্ডার
│   │   ├── 📄 CommunityQA.tsx   # সোশ্যাল ফোরাম
│   │   └── 📄 AdminPanel.tsx    # কন্ট্রোল প্যানেল
│   ├── 📄 App.tsx              # সেন্ট্রাল অ্যাপ রেনডারিং ফাইল
│   └── 📄 index.css            # গ্লোবাল আরজিবি এবং টাইপোগ্রাফি স্টাইলশিট
│
├── 📂 android/                 # 📱 আমাদের অ্যান্ড্রয়েড সোর্স প্রোজেক্ট (সিঙ্ক করার পর তৈরি হবে)
├── 📄 package.json             # ডিপেন্ডেন্সি ও বিল্ড স্ক্রিপ্টস ম্যানেজার
├── 📄 capacitor.config.ts      # ক্যাডাসিটর কনফিগারেশন (পোর্ট, অ্যাপ নেম ও প্যাকেজ আইডি)
├── 📄 vite.config.ts           # মডার্ন Vite কম্পাইলার সেটিংস
└── 📄 ANDROID_BUILD_GUIDE.md   # সম্পূর্ণ বাংলায় অ্যান্ড্রয়েড বিল্ড নির্দেশিকা
```

## 🚀 লোকাল সেটআপ ও রান করার পদ্ধতি (Local Installation)

প্রোজেক্টটি আপনার কম্পিউটারে রান করতে নিচের সহজ কমান্ডগুলো আপনার টার্মিনাল বা রান উইন্ডোতে ব্যবহার করুন:

### ১. ক্লোন ও ডিপেন্ডেন্সি ইনস্টলেশন:
```bash
# গিটহাব থেকে ক্লোন করুন অথবা ডাউনলোডকৃত ফোল্ডারে প্রবেশ করুন
npm install
```

### ২. ওয়েব ভার্সন লোকালি স্টার্ট করা:
```bash
npm run dev
```
এটি আপনার লোকাল কম্পিউটারে `http://localhost:3000` এ অ্যাপটি চালু করে দিবে।

## 📱 অ্যান্ড্রয়েড অ্যাপ তৈরির দ্রুত কমান্ড নির্দেশিকা (Quick Mobile Build)

আপনি আপনার মোবাইলের জন্য অ্যাপটি বিল্ড ট্র্যাকে নিয়ে যেতে জাস্ট নিচের ৩টি সিকোয়েন্সিয়াল কমান্ড ব্যবহার করবেন:

```bash
# ১. রিঅ্যাক্ট ওয়েব কোডটিকে অপ্টিমাইজড ফাইলে রূপান্তর
npm run build

# ২. অ্যান্ড্রয়েড প্ল্যাটফর্ম প্রোজেক্ট উইন্ডো প্রস্তুত করা (শুধু প্রথমবার)
npm run cap:add

# ৩. যেকোনো কোড পরিবর্তন সরাসরি অ্যান্ড্রয়েড অ্যাপ্লিকেশনে সিঙ্ক করা
npm run cap:sync

# ৪. সরাসরি অ্যান্ড্রয়েড স্টুডিওতে আপনার কোডবেস ওপেন করা
npm run cap:open
```
*বিস্তারিত জানতে প্রোজেক্টের ভিতরে থাকা [ANDROID_BUILD_GUIDE.md](./ANDROID_BUILD_GUIDE.md) ফাইলটি পড়ুন।*

## 🔒 সিকিউরিটি ও কাস্টমাইজেশন (Security Settings)
- **পরিবেশ ভেরিয়েবল**: আপনার নিজস্ব API কী এবং মেটাডাটা সেটআপ করার জন্য মূল রোডমেপে `.env.example` ফাইল দেওয়া হয়েছে। এটিকে রিনেম করে `.env` নাম দিয়ে আপনার সিক্রেট কী বসিয়ে নিন।
- **প্যাকেজ আইডি**: প্লে-স্টোরে রিলিজের ক্ষেত্রে `capacitor.config.ts` এর `appId: 'com.iloveyoubd.neonpro'` পরিবর্তন করে আপনার প্রতিষ্ঠানের ডোমেইন রিভার্স অনুসারে নাম দিতে পারেন।
